# 기능정의

## 로그인(id, password) 
1. 구글 로그인 api 적용

## 회원가입
1. (id, password, email, nickname) 구성

## MainPage
1. 공유에디터 저장시 md형식으로 저장
2. 에디터 실시간 공유
3. 파일을 local에 저장 or 게시물 업로드(Like Facebook List)
4. 파일IO(사진저장), OCR기능으로 사진 Text를 표현
5. 보관처리 - 폴더 형식으로 저장
6. 사용자 표시 - 에디터나 파일 등 변경사항 알림기능
7. SNS와 같이 팔로우 and 초대기능(이메일링크) 구현 - 암호화
